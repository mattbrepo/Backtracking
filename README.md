# Backtracking
Generic [Backtracking function](https://en.wikipedia.org/wiki/Backtracking) in Haskell. [N-Queens problem](https://en.wikipedia.org/wiki/Eight_queens_puzzle) used as a test.

**Language: Haskell**

**Start: 2014**

## Example

ghci backtracking.hs

```
*Main> queensN1 8
[1,5,8,6,3,7,2,4]
```
